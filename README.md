# 🎬 Save Studios

Landing page moderna e estratégica para uma marca de audiovisual focada em **storytelling, identidade visual e produção criativa**.

O projeto foi desenvolvido com foco em **design premium**, **alta conversão** e **experiência do usuário**.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Objetivo do projeto

Criar uma landing page com padrão profissional de agência, destacando:

- Autoridade da marca
- Clareza na comunicação
- Design impactante
- Foco em conversão

---

## ✨ Funcionalidades

- Design moderno estilo agência
- Layout responsivo (mobile + desktop)
- Animações suaves
- Seções bem estruturadas (hero, serviços, sobre, contato)
- Call To Action (CTA) estratégico

---

## 🎨 Destaques de UI/UX

- Paleta escura com detalhes em azul (visual premium)
- Tipografia limpa e legível
- Hierarquia visual bem definida
- Experiência fluida na navegação
